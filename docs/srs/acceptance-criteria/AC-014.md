# AC-014

## Criterion

A paused live quiz resumes with adjusted timing so the remaining quiz duration is preserved.

## Linked Requirements

- FR-LQ-006 (manual control)
- BR-008 (pause duration)

## Verification Steps

1. Admin creates a session with `time_per_question = 30s` and 10 questions (total 300s).
2. Admin starts the session at `T0`.
3. Capture `started_at = T0` and `expires_at = T0 + 300s`.
4. Admin pauses at `T0 + 60s`. `paused_at = T0 + 60s`.
5. Wait 90 seconds without resuming.
6. Admin resumes at `T0 + 150s`.
7. Re-read the session row.

## Expected Result

After resume:

- `started_at` is unchanged (`T0`).
- `expires_at` shifts to `T0 + 300s + 90s = T0 + 390s` (original + pause duration).
- `paused_at` is cleared.
- Per-question time taken so far is preserved.

The remaining active time matches the original 300s budget regardless of how long the pause lasted.
