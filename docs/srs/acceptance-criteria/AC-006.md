# AC-006

## Criterion

A live quiz answer is saved with correctness, time taken, and authoritative scoring.

## Linked Requirements

- FR-QUIZ-008 (fetch live question JIT)
- FR-QUIZ-011 (submit secure answer)
- FR-QUIZ-012 (track time taken)
- FR-QUIZ-013 (return correctness)

## Verification Steps

1. Join a live quiz as a student.
2. Open question 1.
3. Wait a few seconds before answering.
4. Pick an option and submit.
5. In the admin Tracking modal for this player, confirm the row shows the player's answer, correctness, and `time_taken` in seconds.
6. Pick a wrong answer for the next question. Confirm it logs as incorrect.
7. After finishing, inspect `kuis_results` rows for this player.

## Expected Result

Each row stores: player ID, question ID, submitted answer, `is_correct`, `time_taken`, and `created_at`. Score is recomputed server-side on every submit (subtract old, add new) so re-answering in standard mode does not double-count.
