# AC-011

## Criterion

An admin can configure category visibility and save it persistently.

## Linked Requirements

- FR-SET-001 (view category tree)
- FR-SET-002 (expand/collapse)
- FR-SET-003 (manage visibility)
- FR-SET-004 (save visibility)

## Verification Steps

1. Sign in as admin. Open Settings tab.
2. Confirm the SettingsMapelTree renders with three pill buttons per row (Hidden / Admin only / Visible).
3. Expand a MAPEL → BAB → Sub-bab path.
4. Change a Sub-bab from `Visible` to `Admin only`.
5. Tap `Save`.
6. Confirm a `success` toast appears: "Settings saved. Changes appear on next page load."
7. Reload the page.
8. Confirm the changed pill state persists.
9. As a separate admin, open the Questions tab. Confirm the admin-only Sub-bab still appears in the filters.
10. As a student, open `/`. Confirm the admin-only Sub-bab is absent from the dropdown.

## Expected Result

Visibility changes persist in `app_settings` via `saveVisibilitySettingsAction`. Toast feedback appears on success and on failure. Hidden rows disappear from student dropdowns; admin-only rows are gated to admin tools only.
