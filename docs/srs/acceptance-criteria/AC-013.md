# AC-013

## Criterion

An admin can create, schedule, start, pause, resume, finish, and delete live quiz sessions.

## Linked Requirements

- FR-LQ-001 (create session)
- FR-LQ-005 (auto-start scheduling)
- FR-LQ-006 (manual control)
- FR-LQ-008 (delete session)
- BR-007 (status lifecycle)
- BR-008 (pause duration)

## Verification Steps

1. Sign in as admin. Open Quiz tab.
2. Tap `Buat sesi`. Pick MAPEL/BAB/Sub-bab, count, time per question, total time, mode.
3. Optionally set an auto-start datetime.
4. Save. Confirm a 6-digit code appears.
5. Tap `Mulai`. Status flips to `active`. `started_at` and `expires_at` are set.
6. Tap `Pause`. Status flips to `paused`. `paused_at` is set.
7. Wait 30 seconds. Tap `Resume`.
8. Confirm `expires_at` shifts forward by ~30 seconds (BR-008).
9. Tap `Selesai`. Status flips to `finished`. `finished_at` is set.
10. Tap delete on a finished session. Confirm it disappears.

## Expected Result

Each transition persists via the appropriate RPC (`set_quiz_active`, `set_quiz_paused`, `set_quiz_resumed`, `set_quiz_finished`, `delete_quiz_session`). The admin UI reflects the new status immediately. Auto-start fires within ~1 minute of `expires_at` via pg_cron.
