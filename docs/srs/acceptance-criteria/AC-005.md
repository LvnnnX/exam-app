# AC-005

## Criterion

A student client cannot access hidden quiz question IDs or correct answers from public lookups.

## Linked Requirements

- NFR-SEC-003 (safe public reads)
- NFR-SEC-004 (correct answer protection)
- BR-010 (correct answers)

## Verification Steps

1. Open the live quiz join page.
2. Inspect the network panel during code lookup.
3. Confirm the response from `public_kuis_logs` includes only safe fields (code, status, mode, total, time, expires_at) and excludes `question_ids`.
4. After joining, inspect the question payload from `get_live_quiz_question`.
5. Confirm the payload is scrambled and, after unscrambling, contains the question body, options A–E, and type only — no `correct_answer` and no `short_answer`.
6. Inspect localStorage. Confirm secrets are encrypted blobs, not plain JSON.

## Expected Result

`question_ids`, `correct_answer`, and `short_answer` never reach the browser. Public views and student-facing RPCs strip these columns. Local storage is encrypted via `lib/security.ts`.
