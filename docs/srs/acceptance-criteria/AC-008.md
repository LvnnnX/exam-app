# AC-008

## Criterion

An unauthenticated user cannot access admin functions.

## Linked Requirements

- FR-ADMIN-001 (login)
- NFR-SEC-002 (admin authorization)
- NFR-SEC-007 (RLS compatibility)

## Verification Steps

1. Open `/admin` in a private browsing window with no `admin_session` cookie.
2. Confirm the AdminLoginView appears.
3. Try direct URLs (`/admin#questions`, `/admin#quiz`, etc.) with no session.
4. Try calling an admin server action via curl with no token.
5. Try calling an admin RPC via the Supabase REST endpoint with no auth header.

## Expected Result

The browser sees only the login screen. Direct URL hits resolve to the same login screen. Server actions reject the call with `Unauthorized`. Admin RPCs reject with RLS / permission errors. No admin data leaks.
