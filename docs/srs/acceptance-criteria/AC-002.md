# AC-002

## Criterion

A student can complete a standard self-paced exam, jump between questions, mark doubtful questions, submit, and view recap.

## Linked Requirements

- FR-EXAM-002 (navigation picker)
- FR-EXAM-015 (standard navigation)
- FR-EXAM-016 (doubtful flag)
- FR-EXAM-017 (submit confirmation)
- FR-QUIZ-019 (daftar soal modal — same component reused)
- FR-EXAM-021 / 022 / 023 (submit, score, recap)

## Verification Steps

1. Open `/`.
2. Step 1: name + Exam + Standard + categories + time limit + question count.
3. Confirm Step 2 header shows `Standard` for navigation.
4. Start exam. Runtime shows action bar with `Back`, `Ragu-ragu`, `Next` / `Finish`.
5. Tap `Daftar soal`. Modal opens with legend dots (black answered, yellow ragu, neutral kosong) and the current tile ringed.
6. Tap a future tile. Runtime jumps to that question. Selected answers persist.
7. On any question, tap `Ragu-ragu`. Pill turns yellow. Re-tap to clear.
8. Navigate Back. Selected answer for the previous question is still selected.
9. On the last question, label flips to `Finish`. Confirm modal copy lists unanswered + doubtful counts.
10. Confirm submit. Score and recap views render correctly.

## Expected Result

Free navigation works. Answers and doubt flags persist across navigation, page refresh (via encrypted localStorage), and into the recap. The recap view marks doubtful questions with a `Ragu-ragu` chip alongside the answer-tone tint.
